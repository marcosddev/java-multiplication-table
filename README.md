# Java Multiplication Table Generator

Este projeto consiste em uma aplicação simples de console desenvolvida em Java. O foco da implementação foi a manipulação de estruturas de repetição (for) e o gerenciamento de estado de variáveis durante iterações consecutivas.

## Objetivo do Projeto
O propósito principal deste desenvolvimento foi aplicar na prática o uso do laço `for`. O sistema não apenas executa operações matemáticas repetitivas, mas também demonstra o conceito de acumulação de valores em variáveis externas ao escopo do loop.

## Funcionalidades Implementadas
O código processa um número base definido na variável inicial e executa as seguintes etapas:
* **Geração de Tabuada:** Utiliza um laço de repetição definido para iterar de 1 a 12, multiplicando a variável de iteração pelo número base e formatando a saída no console.
* **Acumulação de Estado:** A cada ciclo do loop, o produto da multiplicação atual é somado à variável global `sum`, garantindo que o programa mantenha um registro cumulativo de todas as operações realizadas.
* **Saída Estruturada:** O programa exibe a tabuada completa linha por linha e, ao finalizar o ciclo de repetição, apresenta o somatório total de todos os resultados gerados.

## Tecnologias Utilizadas
* Java
